## 3. النظام الشبكي والمسافات

| الرمز | القيمة | الاستخدام |
|-------|--------|-----------|
| `xs` | 4px | تباعد داخلي صغير، أيقونات |
| `sm` | 8px | تباعد بين الأيقونات والنص |
| `md` | 16px | الحشوات الافتراضية داخل الأزرار والبطاقات |
| `lg` | 24px | التباعد بين الأقسام |
| `xl` | 32px | المسافات الكبيرة في التخطيطات الواسعة |
| `2xl` | 48px | الهوامش الكبرى أو الأبطال البصرية |

أنصاف الأقطار المعتمدة:
- `sm = 4px` للعناصر التفاعلية الصغيرة.  
- `md = 8px` للأزرار والبطاقات.  
- `lg = 12px` للحاويات الكبيرة.  
- `pill = 999px` للبادجات أو الأزرار الدائرية.

الظلال:
- `subtle`: لمستوى ارتفاع خفيف على الخلفيات الفاتحة.  
- `medium`: للبطاقات الأساسية.  
- `focus`: ظلّ مخصص لحالات التركيز (Focus) باستخدام لون Primary.

## 4. التراص والشبكة (Layout)
- الشبكة الأساسية: 12 عمودًا بعرض حاوية أقصى 1280px.  
- استخدام `24px` كتباعد أفقي بين البطاقات.  
- تجميع المحتوى في أقسام رئيسية واضحة مع عناوين بارزة.

### 4.1 دعم RTL
- الاتجاه الافتراضي بالعربية: `direction: rtl`.  
- استخدم الخصائص المنطقية (`margin-inline-start`) بدل `margin-left/right` لضمان انعكاس الواجهة.  
- الأيقونات الاتجاهية (مثل السهام) يجب أن تدور تلقائيًا باستخدام CSS logical properties أو استيراد SVGs بدعم الاتجاه.

## 5. المكوّنات الأساسية

### 5.1 الزر الأساسي (Primary Button)
```
background-color: var(--color-brand-primary);
color: var(--color-text-on-brand);
padding: 12px 20px;
border-radius: 8px;
font-weight: 600;
transition: all 0.2s ease;
```
- حالة التحويم: التدرج `linear-gradient(135deg, #326FAE 0%, #2C74CC 100%)`.  
- حالة التعطيل: خلفية `rgba(172, 204, 227, 0.6)` مع نص `var(--color-text-muted)`.  
- مساحة الضغط القابلة للنقر ≥ 44px ارتفاعًا.

### 5.2 الحقول النصية
- حدود 1px بلون `#9CA3AF`.  
- حالة التركيز: تظليل `shadow.focus`.  
- رسائل التحقق تستخدم اللون الأحمر مع أيقونة خطأ واضحة.

### 5.3 البطاقات (Cards)
- خلفية `#F9FAFB`.  
- حدود اختيارية 1px بلون Gray 200.  
- ظل `shadow.subtle`.  
- تباعد داخلي 24px.  
- رؤوس البطاقات تستخدم الخط المتوسط بحجم 1.125rem.

## 6. الموارد التنفيذية

### 6.1 ملف Theme المركزي
المصدر: `frontend/src/styles/theme.ts`

يوفر:
- كائن `theme` متكامل للألوان، الأنماط، والاتجاه.  
- دالة `toCssVariables(theme)` لتحويل القيم إلى CSS Variables.  
- كائن `rtl` لتسهيل التعامل مع الاتجاهات المنطقية.

### 6.2 مكون الشعار
المصدر: `frontend/src/components/Logo.tsx`

خصائص أساسية:
- `size`: التحكم بحجم الحاوية (افتراضي 120px).  
- `variant`: `primary` أو `inverse` لضبط خلفية الشعار.  
- `showWordmark`: إظهار أو إخفاء النص المصاحب.  
- يدعم إدخال خصائص `<img>` الاعتيادية (alt، loading، إلخ).

## 7. دمج النظام في المشاريع
1. استيراد `theme` و`toCssVariables`، وتحميل القيم في `:root` أو ThemeProvider خاص.  
2. تضمين خط **Tajawal** عبر Google Fonts أو استضافة ذاتية.  
3. تطبيق خصائص RTL تلقائيًا بناء على لغة المستخدم (عربي ↔ إنجليزي).  
4. إعادة استخدام مكون `Logo` في الهيدر، السايدبار، والشاشات المصغرة.
